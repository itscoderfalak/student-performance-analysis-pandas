# Student Performance Analysis with Pandas

This project demonstrates a complete data analysis workflow using **Pandas** (no visualization libraries).  
It covers **basic, intermediate, and advanced Pandas operations** to provide insights from a student dataset.

## Features
- Dataset creation and cleaning
- Data exploration and summary statistics
- Filtering, sorting, and grouping
- Conditional columns and performance categorization
- Ranking and pivot tables
- Advanced functions like `apply()`, `query()`, and multi-condition analysis
- Saved final dataset for further use

## Dataset
The dataset contains information about students:
- Name
- Age
- Class
- Marks in Math, Science, and English
- Study hours per week
- Sports
